# PetCard Simulator Sample

ペットカードのシミュレーションUIを確認するための簡易版Reactアプリケーションです。

## 開発メモ

### ビルド方法

```bash
npm run build
```

### ローカルサーバーで確認（オプション）

```bash
npm install -g serve
serve dist
```

## GitHub Pages での公開

このアプリは GitHub Pages にて以下の URL で公開されています：

https://t2-takashima.github.io/petcard-simulator-sample/

デプロイ方法：

```bash
npm run deploy
```
